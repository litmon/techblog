Hugoを使ってブログを構築してみた話の続きです。

### Hugoのテーマ作成

まず、Hugoのテーマのテンプレートを作成します。
コマンドライン上で以下のコマンドを入力しましょう。

```
$ hugo new theme <テーマ名>
```

これだけでテーマが完成しました。簡単！
このテーマを実際に適用するには、config.tomlに以下の記述をするだけです。

```go:config.toml
// ~~~
theme = "テーマ名"

```

これでおしまいです！お疲れさまでした！
